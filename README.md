# Custom GitHub Actions Repository

A collection of public, reusable GitHub Actions for continuous integration and continuous deployment (CI/CD) workflows.

## Available Actions

### 🚀 [Dokploy Deploy](file:///Users/fongfan999/github_actions/dokploy-deploy/README.md) (`dokploy-deploy`)
Builds a Docker image, pushes it to GitHub Container Registry (GHCR), caches it using GitHub Actions cache, and triggers a Dokploy deployment webhook with the head commit's ref and SHA.

- **Directory:** [`./dokploy-deploy`](file:///Users/fongfan999/github_actions/dokploy-deploy/)
- **Configuration & Usage:** See the [Dokploy Deploy Documentation](file:///Users/fongfan999/github_actions/dokploy-deploy/README.md) for full setup instructions and workflow examples.
